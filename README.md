# Portafolio de Derecho Informático

Sitio web estático de portafolio académico para el curso de **Derecho Informático** — Ingeniería en Sistemas de Información, Universidad Mariano Gálvez de Guatemala (UMG), Ciclo 2026.

## Contenido

El portafolio cubre tres módulos del curso:

- **Módulo I** — Fundamentos del Derecho Informático: Internet, DNS, ICANN, dominios `.gt`, relación Derecho-Informática, marco normativo guatemalteco (Decretos 47-2008, 57-2008, 33-98).
- **Módulo II** — Propiedad Intelectual, CLUF, Licencias de software, Firma Digital y criptografía asimétrica.
- **Módulo III** — Comercio Electrónico, Habeas Data, Ciberseguridad en Guatemala, Delitos Informáticos.

## Tecnologías

HTML · CSS · JavaScript — sin dependencias ni proceso de compilación.

## Ver el sitio

Abre `index.html` directamente en cualquier navegador.
